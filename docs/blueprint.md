# **App Name**: Fanuli Carni Equine

## Core Features:

- Homepage Structure: Creation of the basic structure of the home page (Navbar, Hero Section, Values Section, and Footer) using Next.js and Tailwind CSS.
- Navbar: Classic Navbar with links to: Home, I Nostri Prodotti, Salute, Contatti.
- Hero Section: A hero section to display a main title and subtitle.
- Values Section: A section highlighting three core values: Selezione Pugliese, Maestri Macellai, and Gusto e Salute.
- WhatsApp Booking Button: Button to redirect to whatsapp chat and facilitate quick reservations.
- Footer: A footer to display the business address and important legal links.

## Style Guidelines:

- Primary color: Dark Wine Red (#800020) to evoke the color of meat and a sense of elegance.
- Background color: Light Beige/Cream (#FDFBF7) to provide warmth.
- Text color: Dark Brown or Anthracite Gray to provide contrast and readability.
- Titles font: 'Playfair' serif font for all titles, providing elegance and tradition.
- Body font: A clean sans-serif font for long texts to ensure readability.
- Classic and warm layout to evoke trust, history, and craftsmanship.
